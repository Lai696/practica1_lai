Practica 1 - TiendaApp

Esta carpeta ya trae el proyecto tiendaapp con el archivo productos.html modificado.

Para subirlo al repo:
1. Copia la carpeta practica_1_lai dentro de tu repositorio del curso.
2. En la terminal, entra a tu repositorio.
3. Ejecuta:
   git status
   git add practica_1_lai
   git commit -m "feat(practica1): agregar TiendaApp con badges"
   git push

Despues de correr la app y tomar la captura:
1. Guarda la captura como practica_1_lai/evidencia.png
2. Ejecuta:
   git add practica_1_lai/evidencia.png
   git commit -m "docs(practica1): agregar evidencia"
   git push
